Overview

This project started back in Summer 2019. This was my first project ever.
I had to create a static webpage for a company. That was the point in my life, where I officially choose to go this way.
Back then, I just created a folder and put everything inside it. I thought it would be a great idea to make a new webpage for the company.

How we did it?

That project was the one, that I was the proudest of back in 2019.
But time passed on, and new trends came in. If you own a big company, you need to follow today's trends to be the first in the market.

This Project only took a few hours to complete.
We created the new design in about an hour, and the next day I started to develop the page.

Because it's a simple static website it didn't take much time to finish. Of course, jQuery and Bootstrap were the biggest helper.
